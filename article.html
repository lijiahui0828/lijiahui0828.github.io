Introduction¶

The data is a reuters data folder with 21 sgm files and each one has 1,000 articles' information in it, so there are 21,000 articles for us to build the classification model.

The response variable is the topic of the articles, with three categories, representing if the topic is 'earn', 'acq', or another category.

Our goal is to train a model to classify which topic does the article belong to.

def parse_reu(reu):
    """Parses the etree object and returns a list of dictionary of reuters attr
    Output: {'topics': the topic of the article, 'places': where it is located, 
        'split': training/test split, 'body':the text of the article as a set of words with stopwords removed}
    """
    root= reu.getroot()
    articles = root.body.getchildren()
    stop_words = set(stopwords.words('english'))
    reu_pl = []
    for a in articles:
        reu_parse = {}
        if a.attrib['topics'] != 'YES':
            next
        topics = a.find('topics').findall('d')
        if topics:
            reu_parse['topics'] = [t.text for t in topics]
        else:
            reu_parse['topics'] = []
        places = a.find('places').findall('d')
        #if places:
        reu_parse['places'] = [t.text for t in places]
        reu_parse['split'] = a.attrib['lewissplit']
        rtxt = a.find('text')
        word_tokens = word_tokenize(rtxt.text_content())
        filtered_sentence = set([w.lower() for w in word_tokens if not w in stop_words])
        reu_parse['body'] = filtered_sentence
        reu_pl.append(reu_parse)
    return reu_pl

def roc_pr(test_ifearn, earn_score):
    fpr,tpr,threshold = roc_curve(test_ifearn, earn_score,pos_label=1) 
    roc_auc = auc(fpr,tpr)   
    lw = 2

    plt.figure(figsize = (15,8))
    plt.subplot(121)
    plt.plot(fpr, tpr, color='darkorange',  lw=lw, label='ROC curve (area = %0.3f)' % roc_auc)   
    plt.plot([0, 1], [0, 1], color='navy', lw=lw, linestyle='--')  
    plt.xlim([0.0, 1.0])  
    plt.ylim([0.0, 1.05])  
    plt.xlabel('False Positive Rate')  
    plt.ylabel('True Positive Rate')  
    plt.title('ROC for Precomputed Kernel')  
    plt.legend(loc="lower right")  

    precision, recall, thresholds = precision_recall_curve(test_ifearn, earn_score,pos_label=1)
    average_precision = average_precision_score(test_ifearn, earn_score)

    plt.subplot(122)
    plt.step(recall, precision, color='b', alpha=0.2,where='post')
    plt.fill_between(recall, precision, step='post', alpha=0.2,color='b')
    plt.xlabel('Recall')
    plt.ylabel('Precision')
    plt.ylim([0.0, 1.05])
    plt.xlim([0.0, 1.0])
    plt.title('PR curve for Precomputed Kernel: AP={0:0.3f}'.format(average_precision))

Conclusion¶

  • With the kernel matrix based on the overlap of words, which is calculated by tf-idf, we can predict 'earn' with AUC = 0.954, and AP(average precision) = 0.703.

  • Including the overlap of places into the kernel matrix, the AUC of prediction is 0.957 and AP is 0.726. Both AUC and AP are larger than those of prediction with kernel matrix only including overlap of body, so include the information of the places can make the prediction better.

  • Then we also try the linear, quadratic, cubic, rbf and sigmoid kernel. The corresponding AUC and AP are showed below:

Kernel AUC AP
Precomputed 0.954 0.703
Linear 0.954 0.703
Quadratic 0.962 0.818
Cubic 0.958 0.786
Rbf 0.963 0.829
Sigmoid 0.961 0.833

We can find that

  • The linear kernel is the same as our precomputed one.

  • The quadratic kernel is much better then the linear one, but the cubic model is worse than the quadratic one. So for the polynomial kernel, finding a proper degree is important.

  • The Rbf kernel is the one with largest AUC, which is 0.963, and its AP is 0.829.

  • The Sigmoid kernel is the one with largest AP, which is 0.833, and its AUC is 0.961.

  • From the AUC's and AP's, I think the Rbf kernel may be the best.
